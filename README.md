Enter an aritmetic expression to receive its respective solution.

The following symbols may be used:
- '+' addition
- '-' subtraction
- '*' multiplication
- '/' division
- '^' exponentiate
- '(' ')' parentheses
- '[' ']' brackets

Type "quit" to exit the program.

Complie with "javac **/*.java"
Run with "java Calculator" to begin user prompting
- or "java Calculator <filename>" to load a list of expressions at the start
